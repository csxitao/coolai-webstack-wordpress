# 部署指南

## 文件说明

| 文件 | 服务器路径 | 说明 |
|------|-----------|------|
| `coolai-integration.php` | `/wp-content/mu-plugins/` | 后端 API |
| `coolai-nav.html` | `/wp-content/themes/justnews/` | 前端页面 |

## 部署步骤

### 1. 上传后端文件

将 `mu-plugins/coolai-integration.php` 上传到：
```
/wp-content/mu-plugins/coolai-integration.php
```

> mu-plugins 目录下的插件会自动激活，无需手动启用

### 2. 上传前端文件

将 `themes/justnews/coolai-nav.html` 上传到：
```
/wp-content/themes/justnews/coolai-nav.html
```

### 3. 验证部署

1. 访问 `https://你的域名/coolai-nav.html`
2. 测试添加网站功能
3. 测试分类右键菜单
4. 登录状态下测试云同步

## 更新说明

每次更新只需替换对应文件：
- 前端改动 → 替换 `coolai-nav.html`
- 后端改动 → 替换 `coolai-integration.php`

## 故障排查

| 问题 | 解决方案 |
|------|---------|
| 页面空白 | 检查文件是否上传到正确路径 |
| API 报错 | 确认 mu-plugins 目录存在且有写权限 |
| 同步失败 | 检查 WordPress 登录状态 |
| 图标不显示 | 清除浏览器缓存后刷新 |
